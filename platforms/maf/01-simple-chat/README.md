# MAF - Simple Chat

Agente básico de chat usando Microsoft Agent Framework.

## Descripción
Implementación mínima de un agente conversacional.

## Ejecución
```bash
# Instalar dependencias (desde la raíz del proyecto)
cd ../../..
pip install -r requirements.txt

# Ejecutar
cd platforms/maf/01-simple-chat
python src/main.py
```

## Tests
```bash
pytest tests/
```
