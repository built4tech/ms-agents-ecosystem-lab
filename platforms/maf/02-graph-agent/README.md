# MAF - Graph Agent

Agente con integración de Microsoft Graph usando Microsoft Agent Framework.

## Descripción
Agente capaz de interactuar con Microsoft Graph API para acceder a datos de calendario, correo, archivos, etc.

## Ejecución
```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
python src/main.py
```

## Tests
```bash
pytest tests/
```
