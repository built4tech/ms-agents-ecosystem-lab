# CrewAI - Simple Chat

Agente básico de chat usando CrewAI.

## Descripción
Implementación mínima de un agente conversacional con modelos en Azure/OpenAI/Ollama.

## Ejecución
```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
python src/main.py
```

## Tests
```bash
pytest tests/
```
