# CrewAI - Simple Chat

Agente básico de chat usando CrewAI.

## Descripción
Implementación mínima de un agente conversacional con modelos en Azure/OpenAI/Ollama.

## Ejecución
```bash
# Instalar dependencias (desde la raíz del proyecto)
cd ../../..
pip install -r requirements.txt

# Ejecutar
cd platforms/crewai/01-simple-chat
python src/main.py
```

## Tests
```bash
pytest tests/
```
