# CrewAI - Orchestrated

Sistema multi-agente orquestado usando CrewAI.

## Descripción
Implementación de múltiples agentes coordinados para tareas complejas.

## Ejecución
```bash
# Instalar dependencias (desde la raíz del proyecto)
cd ../../..
pip install -r requirements.txt

# Ejecutar
cd platforms/crewai/03-orchestrated
python src/main.py
```

## Tests
```bash
pytest tests/
```
